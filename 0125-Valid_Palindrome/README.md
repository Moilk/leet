## 题目描述

给定一个字符串，验证它是否是回文串，只考虑字母和数字字符，可以忽略字母的大小写。

**说明：**本题中，我们将空字符串定义为有效的回文串。

**示例 1:**

&emsp;**输入:** `"A man, a plan, a canal: Panama"`  
&emsp;**输出:** `true`

**示例 2:**

&emsp;**输入:** `"race a car"`
&emsp;**输出:** `false`
