# 5. 最长回文子串

## 题目描述

给定一个字符串 `s`，找到 `s` 中最长的回文子串。你可以假设 `s` 的最大长度为 1000。

**示例 1：**

&emsp;**输入:** "babad"  
&emsp;**输出:** "bab"  
&emsp;**注意:** "aba" 也是一个有效答案。  

**示例 2：**

&emsp;**输入:** "cbbd"  
&emsp;**输出:** "bb"
